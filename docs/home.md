# Home Page

Welcome to **CoffeeShop**, a better [CoffeeScript][1] shell.

You will need a V8 browser (Chromium, Chrome, Opera) to use this shell. Look
through the [the Book](/docs/front.md) to get started, at least looking at
the Shortcuts page.
